#############################
Document's Retrieval System
#############################


#############################################################################

Information



This system was designed during "Information Retrieval" course in BGU 2020.
The system is designed to get a path to the corpus of the documents,
and a path to the posting files directory.
The system indexes the documents inside the corpus and save the results to the
posting files in the directory specified.

Beta version 1.0
- released on 20.12.19

#############################################################################

Usage



- Run the .bat file
- Insert a path to the corpus directory
- Insert a path to the posting files destination directory
- Click "Start"
(DO NOT CLOSE THE APP DURING THE INDEXING PROCCESS)

For loading the dictionary from posting files:
- Insert a path to the posting files destination directory
- Click "Load"

Click "Show Dictionary" for showing details of the dictionary
after the indexing proccess.

Stemming is enabled by checking the "Stemming" button.

Deleting the posting files and clearing the dictionary is done by "Reset" button.
